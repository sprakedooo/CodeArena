SYSTEM CONCEPT: GAME-BASED ADAPTIVE LEARNING WITH ML
================================================================================

KEY FEATURES:
1. Language Selection     - Students choose Python, Java, or C++
2. Adaptive Questions     - Difficulty adjusts based on performance
3. Reward System          - Points, badges, and level progression
4. Rule-Based AI Hints    - Generated when answers are wrong
5. Weak Area Tracking     - System identifies struggling topics
6. Progress Levels        - Beginner → Intermediate → Advanced
7. Decision Tree ML       - Predicts optimal learning paths

MACHINE LEARNING COMPONENT:
- Model Type: Decision Tree Classifier
- Features: accuracy, streak, questions answered, current level
- Predictions:
  * Difficulty Recommendation (INCREASE/MAINTAIN/DECREASE)
  * Intervention Detection (NEEDS_HELP/AT_RISK/ON_TRACK)
  * Topic Priority (FOCUS_WEAK/EXPLORE_NEW/REINFORCE)

ML API ENDPOINTS:
- GET /api/ml/predict/:userId/:language     - All predictions
- GET /api/ml/difficulty/:userId/:language  - Difficulty only
- GET /api/ml/intervention/:userId/:language - Intervention check
- GET /api/ml/model-info                    - Model documentation
- POST /api/ml/analyze                      - Custom data analysis

LEARNING FLOW:
1. Student logs in
2. Selects a programming language
3. Views ML insights on dashboard
4. Starts at Beginner level
5. Answers questions (game-style)
6. Correct = Points + ML checks for level advancement
7. Wrong = AI hint + weak area recorded
8. After session = AI feedback on weak areas
